## ๐ Quick Start (2 minutes)

1. **Get API key** (free from https://fixer.io)
2. **Set environment:**
   ```bash
   echo "CURRENCY_API_KEY=your_key" >> .env
   ```
3. **Restart server:**
   ```bash
   python pos_server.py
   ```
4. **Test in browser:**
   - Add items to cart
   - Select "Cash" tender
   - EUR panel appears automatically
